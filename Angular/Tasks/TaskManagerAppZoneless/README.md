# Task Manager

## Overview

Task Manager is an Angular web application that helps users organize and manage their daily tasks efficiently. It provides secure authentication and a simple, user-friendly interface for improving productivity.

## Features

* User registration and login
* Protected routes
* Task creation and management
* Notification service
* Responsive design
* Reusable components

## Technologies Used

* Angular
* TypeScript
* Bootstrap
* RxJS
* Angular Router

## Getting Started

### Installation

```bash
git clone <repository-url>
cd Task-Manager
npm install
ng serve
```
